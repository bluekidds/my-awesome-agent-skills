# My Awesome Agent Skills

A collection of custom Claude skills for automating creative workflows.

## Skills

### anime-pilgrimage

Create interactive anime pilgrimage travel guides with:
- Real-location maps (Leaflet.js, no API key needed)
- Anime vs reality photo comparison sliders
- Multi-day optimized itineraries
- Output as interactive HTML, PDF, or PPTX

**Usage:** Tell Claude something like "Help me create a pilgrimage guide for Your Name" or "I want to visit Weathering With You locations in Tokyo"

**Install:** Download anime-pilgrimage.skill and double-click to install into Claude.
